asyncstart

Initiates a Remote Procedure Call (RPC) without waiting for the response, useful for calls that may take a long time. Responses can be retrieved using the asyncget command.

Input Parameters

Method expects a list of parameters:

  • method name (string): The RPC method to be called asynchronously.
  • param1 (optional): First parameter for the method call.
  • param2 (optional): Second parameter for the method call, and so forth.

Output Format

Returns the number of items currently in the job queue.

Examples

digiasset_core-cli asyncstart "methodName" "param1" "param2"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "asyncstart", "params": ["methodName", "param1", "param2"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
