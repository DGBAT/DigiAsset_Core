getcontract

Gets the chain encoded data for a smart contract.

Input Parameters

  • contractAddress (string): Contract address you want to disable

Output Format

{
   publisher: {
      address: string - publishing address,
      country: string - if kyc enabled country of creator,
      name:    string - if kyc enabled will have name or hash,
      hash:    string - if kyc enabled will have hash or name,
      alias:   string - if an alias is defined and verified will be included
   },
   contractAddress: string - contract address,
   cid:     string - cid of where to find the metadata for the contract,
   version: integer - version number the metadata complies to,
   state:   bool - if contract is currently usable
}
