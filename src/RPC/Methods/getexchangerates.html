getexchangerates

Returns a list of all current exchange rates for different currencies and assets. These rates are specific to certain block heights and can be retrieved for current or past blocks as needed.

Exchange rate history starts at 12642650.

Input Parameters

  • block height (unsigned int, optional): Specify the block height to retrieve historical exchange rates. If omitted, current rates are returned.

Output Format

JSON array of objects, each containing:
{
    "height": integer,     // Block height of the exchange rate
    "address": string,     // Address associated with the exchange rate
    "index": integer,      // Index of the exchange rate
    "value": double        // Value of the exchange rate
}

Note: DGB has an implied value of 100,000,000 sats and is not included in the output.

Examples

digiasset_core-cli getexchangerates

digiasset_core-cli getexchangerates 13000000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getexchangerates", "params": [650000]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
