addressstats

Returns statistics about addresses over time. Note that the last result may not represent a full time period.

Input Parameters

  • start time (integer, optional, default = 0): Start time of the statistics period.
  • end time (integer, optional, default = maximum possible value): End time of the statistics period.
  • time frame (integer, optional, default = 86400): Time frame for statistics, in seconds.

Output Format

JSON array of objects:
[
  {
    "time": integer,          // The end time of the time frame
    "new": integer,           // Number of new addresses created
    "used": integer,          // Number of addresses used
    "withAssets": integer,    // Number of addresses with assets
    "over0": integer,         // Number of addresses with balance over 0
    "over1": integer,         // Number of addresses with balance over 1
    "over1k": integer,        // Number of addresses with balance over 1k
    "over1m": integer,        // Number of addresses with balance over 1m
    "quantumInsecure": integer,// Number of quantum insecure addresses
    "total": integer          // Total number of addresses
  },
  ...
]

Examples

digiasset_core-cli addressstats

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "addressstats", "params": [0, 1609459200, 86400]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
