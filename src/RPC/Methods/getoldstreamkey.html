getoldstreamkey

This command simulates the old DigiAsset Stream and is marked for deprecation. It should not be used for new projects. The function returns data associated with a specified key from the legacy system.

Input Parameters

key (string or integer): The key for which data is requested from the old stream.

Output Format

Depending on the key, the output may vary. Typically, it returns data (string or JSON object) associated with the key.

Examples

digibyte-cli getoldstreamkey "someKey123"

digibyte-cli getoldstreamkey 456

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getoldstreamkey", "params": ["someKey123"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
