getassetdata

Returns detailed data about a specific DigiAsset, which may include IPFS information unless explicitly excluded.

Input Parameters

Two possible usages:

  • Usage 1:
    • assetIndex (integer): The index of the asset.
    • exclude IPFS (boolean, optional, default = false): Whether to exclude IPFS data.
  • Usage 2:
    • assetId (string): The ID of the asset.
    • txid (string, optional): The transaction ID involving the asset.
    • vout (integer, optional): The output number in the transaction.
    • exclude IPFS (boolean, optional, default = false): Whether to exclude IPFS data.

Output Format

JSON object that represents the asset in detailed format. Includes information such as asset name, creator, associated metadata, and IPFS links unless excluded.

Examples

digibyte-cli getassetdata 12345 true

digibyte-cli getassetdata "Labcde12345" "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC" 0 true

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getassetdata", "params": ["Labcde12345", "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC", 0, true]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
