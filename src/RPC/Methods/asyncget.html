asyncget

Retrieves the results of an asyncstart command with the same parameters.

Input Parameters

Method expects a list of parameters:

  • method name (string): Name of the method whose results are to be retrieved.
  • param1 (optional): First additional parameter of the method.
  • param2 (optional): Second additional parameter of the method, and so forth.

Output Format

The output format is the same as the method called, but if the output is an object, an additional cacheTime parameter is included. Returns false if the file does not exist or cannot be read.

Examples

digiasset_core-cli asyncget "methodName"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "asyncget", "params": ["methodName"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
