getdgbequivalent

This RPC command converts fiat or cryptocurrency values to their equivalent in DGB based on exchange rates published on specified addresses. It supports converting values using the latest exchange rates provided by trusted publishing addresses.

Will throw error if sync > 120 blocks behind.

Input Parameters

  • address (string): The address from which the exchange rate is retrieved.
  • index (unsigned int): Index of the rate at the specified address (0 for fiat conversions, others for specific cryptocurrencies).
  • amount (unsigned int): The fixed precision amount to convert, with 8 decimals (e.g., to convert 1 USD to DGB, use 100000000 which represents 1.0 in fixed precision).

Note: Current publishing addresses for exchange rates include:

  • Fiat currency conversions: dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh
  • Cryptocurrency conversions: dgb1qlk3hldeynl3prqw259u8gv0jh7w5nwppxlvt3v

Output Format

Returns the equivalent amount in DGB satoshis.

Examples

# Convert 1 USD to DGB using the fiat currency conversion address:
digiasset_core-cli getdgbequivalent "dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh" 0 100000000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getdgbequivalent", "params": ["dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh", 0, 100000000]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
