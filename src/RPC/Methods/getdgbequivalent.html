getdgbequivalent

This RPC command converts currency values to their equivalent in DGB based on exchange rates published at specified addresses. Each address can store up to 10 different exchange rates, selectable via an index.

Will throw an error if sync is more than 120 blocks behind.

Input Parameters

  • address (string): The address where the exchange rate is published.
  • index (unsigned int): Index of the rate at the specified address, ranging from 0 to 9.
  • amount (unsigned int): The fixed precision amount to convert, with 8 decimals (e.g., 100000000 represents 1.0).

Note: See the default exchange rates and their indices here.

Output Format

Returns the equivalent amount in DGB satoshis.

Examples

Example converting using the first exchange rate at a specified address:

digiasset_core-cli getdgbequivalent "dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh" 0 100000000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getdgbequivalent", "params": ["dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh", 0, 100000000]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
