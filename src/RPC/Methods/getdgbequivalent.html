getdgbequivalent

Returns the equivalent amount in DigiByte satoshis for a given amount of another asset or currency, using a specific exchange rate index.

Input Parameters

  • address (string): The address associated with the asset or currency.
  • index (unsigned int, between 0 and 9): Index specifying which exchange rate to use.
  • amount (unsigned int, 8 decimals): Fixed precision amount of the asset to convert into DigiByte satoshis.

Output Format

Unsigned integer representing the number of DigiByte satoshis equivalent to the input amount.

Examples

digibyte-cli getdgbequivalent "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC" 1 100000000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getdgbequivalent", "params": ["D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC", 1, 100000000]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
