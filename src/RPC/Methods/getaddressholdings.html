getaddressholdings

Returns information related to a specific address, detailing the holdings for each asset. This includes DigiByte, represented as asset index 1, if the storenonassetutxo setting is enabled.

Input Parameters

address (string): The blockchain address for which holdings information is requested.

Output Format

JSON object with asset indexes as keys and quantities as values:
{
    "1": 100000,   // Example for DigiByte if included
    "2": 150,
    "3": 300,
    ...
}

Examples

digiasset_core-cli getaddressholdings "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddressholdings", "params": ["DGb6Bv..."]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
