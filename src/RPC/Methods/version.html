version

This command provides the current version of the DigiByte software running on the node. It's useful for checking compatibility, troubleshooting, or ensuring that the software is up to date.

Input Parameters

No input parameters required for this command.

Output Format

Returns a string:
{
    "version": "x.x.x.x"   // The current version number of the DigiByte software
}

Examples

digiasset_core-cli version

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "version", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
