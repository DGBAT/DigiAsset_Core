debugwaittimes

Returns a string of wait times to help improve performance. The data is extracted from profiling information related to database and core Digibyte system operations.

Input Parameters

No input parameters are needed for this method.

Output Format

JSON array of strings, each representing a profiling log entry:
[
    "Time taken for XYZ operation: 5ms",
    "Execution time for ABC process: 3ms",
    ...
]

Examples

digiasset_core-cli debugwaittimes

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "debugwaittimes", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
