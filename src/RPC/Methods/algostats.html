algostats

Returns mining statistics over a given time period for different algorithms. Note that the last result may not represent a full time period.

Input Parameters

  • start time (integer, optional, default = 0): Start time of the statistics period.
  • end time (integer, optional, default = maximum possible value): End time of the statistics period.
  • time frame (integer, optional, default = 86400 seconds): Time frame for the statistics, representing daily intervals by default.

Output Format

JSON array of objects:
[
  {
    "time": integer,          // The end time of the time frame
    "algo": [
      {
        "min": float,        // Minimum difficulty encountered
        "max": float,        // Maximum difficulty encountered
        "avg": float,        // Average difficulty across the period
        "count": integer     // Number of blocks mined with this algorithm
      },
      null,                  // Placeholder for missing algorithms
      ...
    ]
  },
  ...
]

Examples

digibyte-cli algostats

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "algostats", "params": [0, 1620000000, 86400]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
