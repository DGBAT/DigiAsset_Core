listlastassetspageindexes

Returns a structured list of asset indexes, ordered by issuance height and grouped by page, with optional filtering based on Permanent Storage Pool (PSP) membership. This command is designed to facilitate pagination and efficient data retrieval in applications that need to display asset listings.

Input Parameters

  • numberOfRecords (unsigned int): The number of records to include in each page of results.
  • filter (object, optional): A filter object to refine results based on specific criteria:
    • psp (bool or unsigned int): If true, only returns assets that are part of a PSP; if false, only those that are not part of a PSP; if an integer, only those that are part of the specified PSP.

Output Format

JSON array of objects, each representing a page of asset indexes:
[
    {
        "start": unsigned int,       // The highest asset index in the current page
        "skips": [                    // Array of asset indexes that were skipped over
            unsigned int,
            ...
        ]
    },
    ...other pages...
]

Note: Filtering may result in fewer items than requested and can lead to several pages of empty results.

Examples

digiasset_core-cli listlastassetspageindexes 10

digiasset_core-cli listlastassetspageindexes 10 '{"psp": true}'

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listlastassetspageindexes", "params": [10, {"psp": 1}]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
