sendmany

This command sends multiple DigiByte or DigiAssets payments to one or more addresses or domains in a single transaction. It includes support for domain resolution, converting domain names into DigiByte addresses before processing the transaction.

Input Parameters

  • fromAccount (string): The account from which the funds should be taken. Deprecated and ignored if provided.
  • amounts (object): An object where each key can be a DigiByte address or a domain, and the value is the amount of DGB to send.
  • Optional parameters for transaction crafting, such as minconf (minimum confirmations), comment, subtractfeefrom (array of addresses that will have the fee subtracted from their transaction amount), and other specifics detailed in the Bitcoin Core sendmany command documentation.

Output Format

Returns the transaction ID (txid) of the sent transaction.

Examples

digibyte-cli sendmany "" '{"example.dgb": 0.5, "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC": 1.5}'

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "sendmany", "params": ["", {"example.dgb": 0.5, "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC": 1.5}]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
