getcontractstate

Gets if a contract is active or not and what the requirements the contract needs and has

Input Parameters

  • contractAddress (string): Contract address you want to disable
  • height (integer - optional default current): height you are enquiring about

Output Format

{
   active: bool - if contract is active or not,
   requirements: [
      {
         assetId: string - assetId of requirement,
         assetIndex: integer - asset index of requirement,
         need: integer - number of assets needed,
         have: integer - number of assets contract has
      }
   ]
}
