listmycontractaddresses

Gets a list of contracts operated by a plugin

Input Parameters

  • pluginName (string): Name of the plugin to be used. Must be one of the plugins installed on this node.
  • account (string, optional): If multiple accounts are set up this field allows selecting the one to use. Leave blank if multiple accounts not set

Output Format

[
    "contractAddress1", "contractAddress2"...
]
