syncstate

This command provides the current synchronization state of the DigiByte blockchain and DigiAsset systems. It details how closely the node's data aligns with the blockchain and the operational state of the DigiAsset processor.

Input Parameters

No input parameters required for this command.

Output Format

JSON object containing:
{
    "count": unsigned int,       // The block height DigiByte Core is currently synced to
    "sync": int                  // Sync state:
                                 // Negative values indicate how many blocks behind DigiAsset processing is.
                                 // 0 = Fully synced
                                 // 1 = Stopped
                                 // 2 = Initializing
                                 // 3 = Rewinding
                                 // 4 = Optimizing (occurs during initial sync)
}

Note: A sync state more than 120 blocks behind is considered unsafe for transactions.

Examples

digiasset_core-cli syncstate

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "syncstate", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
