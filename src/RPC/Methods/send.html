send

This command sends DigiByte or DigiAssets to specified addresses or domains. It has been extended to resolve domain names into DigiByte addresses before executing the transaction, aligning with typical wallet functionality but with domain support.

Input Parameters

  • outputs (array): An array of objects, where each object's key can be a DigiByte address or a domain, and the value is the amount of DGB or asset to send.
  • Additional parameters for transaction crafting (optional): Can include fee rate, confirmation target, and other transaction details similar to the Bitcoin Core send command.

Output Format

Returns the transaction ID (txid) of the sent transaction.

Examples

digiasset_core-cli send '[{"example.dgb": 0.5}]'

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "send", "params": [[{"D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC": 1.5}]]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
