listassetissuances

Returns a list of all asset issuances associated with a specific asset ID, detailing each issuance's basic information.

Input Parameters

  • assetId (string): The asset ID for which issuances are to be listed.

Output Format

JSON array of objects, each representing an asset issuance:
[
    {
        "assetIndex": unsigned int,   // Index of the asset
        "txid": string,               // Transaction ID of the issuance
        "amount": unsigned int,       // Amount issued
        "height": int,                // Block height of the issuance
        "cid": string                 // Content identifier for the asset
    },
    ...other issuances...
]

Examples

digiasset_core-cli listassetissuances "assetId12345"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listassetissuances", "params": ["assetId12345"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
