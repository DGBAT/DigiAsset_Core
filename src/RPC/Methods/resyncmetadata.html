resyncmetadata

This command initiates a resynchronization of all IPFS metadata associated with subscribed Permanent Storage Pools (PSPs). It pins all metadata to ensure it is available locally, though the download and pinning process may continue to run after the command returns a response.

Input Parameters

No input parameters required for this command.

Output Format

Returns a boolean value:
true - Indicates that the pinning process has been initiated successfully.

Note: A return value of true does not guarantee that all metadata has been successfully downloaded and pinned, only that the process has begun.

Examples

digiasset_core-cli resyncmetadata

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "resyncmetadata", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
