createcontract

Creates a smart contract using an installed plugin

Input Parameters

  • pluginName (string): Name of the plugin to be used. Must be one of the plugins installed on this node.
  • params (json): parameters needed to build the contract. Needed parameters can be retrieved using the getcontractconstructorparameters command.
  • account (string, optional): If multiple accounts are set up this field allows selecting the one to use. Leave blank if multiple accounts not set

Output Format

string containing created contract address
