shutdown

This command safely shuts down the DigiByte node's subsystems, ensuring that all components such as the blockchain analyzer and IPFS are properly stopped before closure. It logs a critical message indicating it is safe to shut down the system.

Input Parameters

No input parameters required for this command.

Output Format

Returns a boolean value:
true - Indicates that the shutdown process has been initiated successfully.

Examples

digiasset_core-cli shutdown

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "shutdown", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
