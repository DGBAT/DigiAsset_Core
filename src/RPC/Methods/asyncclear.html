asyncclear

Clears the results of an asynchronous call.

Input Parameters

Method expects a list of parameters:

  • method name (string): Name of the method whose results are to be cleared.
  • param1 (optional): First additional parameter of the method.
  • param2 (optional): Second additional parameter of the method, and so forth.

Output Format

Boolean:
true if the results were successfully cleared,
false if there were no results to clear.

Examples

digiasset_core-cli asyncclear "methodName"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "asyncclear", "params": ["methodName"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
