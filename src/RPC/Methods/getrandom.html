getrandom

This command provides a psudo random value for a given transaction output or block. It is designed to be unpredictable but at the same time provable after the fact that the value was not altered by the site operator.

Input Parameters

Two possible usages:

  • Usage 1 (transaction output):
    • txid (string): The txid of the transaction(it must have already been mined with at least 2 confirms)
    • vout (unsigned int): output number
    • outOf (unsigned int 64bit, optional): Will return a number from 0 to 1 less than this value. 2^64 used if undefined
  • Usage 2:
    • blockHeight (unsigned integer): The height of the block in question(it must have already been mined with at least 2 confirms)
    • outOf (unsigned int 64bit, optional): Will return a number from 0 to 1 less than this value. 2^64 used if undefined

Output Format

64bit unsigned integer

Examples

digiasset_core-cli getrandom 19000000 256

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getrandom", "params": [19000000,256]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
