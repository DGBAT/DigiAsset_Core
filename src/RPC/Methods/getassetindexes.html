getassetindexes

Returns a list of asset indexes associated with a specific asset ID. Most assets will have only one index, but this command can be used to retrieve all indexes for assets that have multiple.

Input Parameters

assetId (string): The asset ID for which indexes are requested.

Output Format

JSON array of unsigned integers, each representing an asset index:
[12345, 67890, ...]

Examples

digibyte-cli getassetindexes "Labcde12345"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getassetindexes", "params": ["Labcde12345"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
