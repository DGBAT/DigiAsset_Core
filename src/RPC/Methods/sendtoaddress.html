sendtoaddress

This command sends DigiByte (DGB) or DigiAssets to a specified address or domain. It extends the standard functionality by resolving domain names to DigiByte addresses before processing the transaction, enhancing usability and flexibility.

Input Parameters

  • address (string): The DigiByte address or domain to which the DGB or assets are to be sent.
  • amount (numeric): The amount in DGB to send.
  • Additional optional parameters for transaction customization, such as comment, comment_to, subtractfeefromamount (boolean), replaceable (boolean), conf_target (numeric), estimate_mode (string), providing flexibility similar to the Bitcoin Core sendtoaddress command.

Output Format

Returns the transaction ID (txid) of the sent transaction.

Examples

digiasset_core-cli sendtoaddress "example.dgb" 1.5

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "sendtoaddress", "params": ["example.dgb", 1.5]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
