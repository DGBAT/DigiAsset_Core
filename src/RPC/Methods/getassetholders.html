getassetholders

Returns a list of all addresses holding a specific asset and the quantity they hold.

Input Parameters

assetIndex (unsigned int): The index of the asset for which holder information is requested.

Output Format

JSON object with addresses as keys and the quantity of the asset held as values:
{
    "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC": 1500,
    "Dabc123Def456Ghi789Jkl010Mno234Pqr": 300,
    ...
}

Examples

digiasset_core-cli getassetholders 12345

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getassetholders", "params": [12345]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
