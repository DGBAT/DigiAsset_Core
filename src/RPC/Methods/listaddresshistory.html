listaddresshistory

listaddresshistory

Returns a list of transaction IDs (txids) that involve a specific address, with optional filters for block height and result limit. Note that the results may be limited based on the pruning level of the node.

Input Parameters

  • address (string): The address to retrieve transaction history for.
  • minHeight (unsigned int, optional, default = 1): The minimum block height from which to retrieve transactions.
  • maxHeight (unsigned int, optional, default = infinity): The maximum block height to which to retrieve transactions.
  • limit (unsigned int, optional, default = infinity): The maximum number of transactions to return.

Output Format

JSON array of strings, each representing a transaction ID:
["txid1", "txid2", "txid3", ...]

Examples

digiasset_core-cli listaddresshistory "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC"

digiasset_core-cli listaddresshistory "D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC" 100 200 10

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listaddresshistory", "params": ["D7hrf5D21PV24ksEGRdYCZ6JqgsboSVKKC", 100, 200, 10]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
