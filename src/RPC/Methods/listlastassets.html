listlastassets

Returns a list of asset IDs, starting from the newest issuance and going backwards, optionally filtered by specific criteria. This command is useful for retrieving recent asset issuance data.

Input Parameters

  • numberOfRecords (unsigned int, optional, default = infinity): The maximum number of asset records to return.
  • firstAsset (unsigned int, optional, default = newest): The starting point for listing assets, specified as an index.
  • basic (bool, optional, default = true): Whether to return basic information about each asset or detailed information.
  • filter (object, optional): A filter object to refine the results based on specific criteria like PSP membership:
    • psp (bool or unsigned int): If true, only returns assets part of a PSP; if false, only those not part of a PSP; if unsigned int, only those that are part of the specified PSP.

Output Format

If basic is true:
JSON array of objects, each representing basic asset information:
[
    {
        "assetIndex": unsigned int,
        "assetId": string,
        "cid": string,
        "height": int
    },
    ...other assets...
]

If basic is false:
JSON array of detailed asset information.

Examples

digiasset_core-cli listlastassets 10

digiasset_core-cli listlastassets 10 5 true '{"psp": true}'

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listlastassets", "params": [10, 5, true, {"psp": true}]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
