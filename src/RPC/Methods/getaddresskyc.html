getaddresskyc

Returns KYC (Know Your Customer) information related to a specific address. The information may include the country code and optionally the name or a hash of the name, depending on the level of verification.

Input Parameters

address (string): The blockchain address for which KYC information is requested.

Output Format

JSON object with the following fields:
{
    "address": "DGb6Bv...",   // The requested address
    "country": "USA",         // ISO 3166-1 alpha-3 country code (if KYC verified)
    "name": "John Doe",       // Name of the person (optional, if verified and not anonymous)
    "hash": "abc123def456..." // SHA256 hash of the person's full name and pin (optional, if anonymous)
}

Examples

digiasset_core-cli getaddresskyc "DGb6Bv..."

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddresskyc", "params": ["DGb6Bv..."]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
