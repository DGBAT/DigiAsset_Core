listlastblocks

Returns a list of the most recently processed blocks in the DigiByte blockchain, starting from a specified block or the newest by default. This is useful for tracking block details such as height, hash, time, and the mining algorithm used.

Input Parameters

  • limit (unsigned int): The number of blocks to return.
  • start (unsigned int, optional, default = newest block): The block height from which to start the list.

Output Format

JSON array of objects, each representing a block:
[
    {
        "height": unsigned int,  // Block height
        "hash": string,          // Block hash
        "time": unsigned int,    // Block timestamp
        "algo": string           // Mining algorithm used
    },
    ...other blocks...
]

Examples

digiasset_core-cli listlastblocks 5

digiasset_core-cli listlastblocks 5 100000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listlastblocks", "params": [5, 100000]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
