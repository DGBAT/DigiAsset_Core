getpsp

getpsp

Returns detailed data about a specified Permanent Storage Pool (PSP) based on its index. This is useful for retrieving information about PSPs registered within the DigiByte network.

Input Parameters

index (unsigned int): The index number of the PSP to retrieve data from.

Output Format

JSON object representing the PSP:
{
    "name": "PSP Name",
    "description": "Description of the PSP",
    "url": "Path to PSP website",
    "files": "list of ipfs CIDS thar are part of PSP"
}

Examples

digiasset_core-cli getpsp 1

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getpsp", "params": [1]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
