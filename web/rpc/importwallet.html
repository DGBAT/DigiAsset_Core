importwallet

importwallet "filename"

Imports keys from a wallet dump file (see dumpwallet). Requires a new wallet backup to include imported keys.

Note: Use "getwalletinfo" to query the scanning progress.

Argument #1 - filename

Type: string, required

The wallet file

Result

null    (json null)

Examples

Dump the wallet:

digiasset_core-cli dumpwallet "test"

Import the wallet:

digiasset_core-cli importwallet "test"

Import using the json rpc call:

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "importwallet", "params": ["test"]}' -H 'content-type: text/plain;' http://127.0.0.1:14022/
